# Go Fiber Server

## Bookmark app with support of LLM AI Chat
## Modules used
- Air, for hot reload
- Fiber, for web server
- Viper, for config file
- Testify, for testing
- sqlx, for database
- pgx, for postgreSQL driver
- squirrel, for sql generation
- golang-migrate, for migration
- go-playground/validator, for validation
